# CyberShield AI 🛡️

An AI-powered cyber threat analysis platform that detects phishing emails, malicious URLs, and network log threats using the LLaMA 3.3 70B model.

## Live Demo
https://cyber-threat-analyzer-z596.onrender.com

## Features
- Phishing email detection
- Malicious URL analysis  
- Network log threat analysis
- Color-coded threat levels (Safe to Critical)
- Analysis history

## Tech Stack
- Python + Flask (Backend)
- LLaMA 3.3 70B via Groq API (AI)
- HTML, CSS, JavaScript (Frontend)
- Deployed on Render.com
